Help

Browser Compatibility

To use the translator correctly, it must be opened in Google Chrome version 141 or higher. We apologize for any inconvenience; as soon as the necessary APIs are available for other browsers, you can enjoy the translator in your favorite browser.

Downloading the Translation Model

It may take a few moments to launch as it downloads the translation model from the Chrome Translate API. This model, powered by artificial intelligence, allows for accurate and local translations.

Security and Privacy

Unlike other services, this translator does not use external third-party APIs, which helps protect your personal information. All translations are processed directly from the translator itself, ensuring greater security and confidentiality.
